工具列表:
- **comment**: Comment on an Instagram post
- **explore**: Instagram explore/discover trending posts
- **followers**: List followers of an Instagram user
- **following**: List accounts an Instagram user is following
- **like**: Like an Instagram post
- **profile**: Get Instagram user profile info
- **saved**: Get your saved Instagram posts
- **save**: Save (bookmark) an Instagram post
- **search**: Search Instagram users
- **unfollow**: Unfollow an Instagram user
- **unlike**: Unlike an Instagram post
- **unsave**: Unsave (remove bookmark) an Instagram post
- **user**: Get recent posts from an Instagram user
