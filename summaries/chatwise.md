工具列表:
- **ask**: Send a prompt and wait for the AI response (send + wait + read)
- **export**: Export the current ChatWise conversation to a Markdown file
- **history**: List conversation history in ChatWise sidebar
- **model**: Get or switch the active AI model in ChatWise
- **new**: Start a new conversation in ChatWise
- **read**: Read the current ChatWise conversation history
- **screenshot**: Capture a snapshot of the current ChatWise window (DOM + Accessibility tree)
- **send**: Send a message to the active ChatWise conversation
- **status**: Check active CDP connection to ChatWise Desktop
