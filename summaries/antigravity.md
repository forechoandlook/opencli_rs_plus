工具列表:
- **dump**: Dump the DOM to help AI understand the UI
- **extract-code**: Extract multi-line code blocks from the current Antigravity conversation
- **model**: Get or change the active AI model in Antigravity
- **new**: Start a new Antigravity conversation
- **read**: Read the current Antigravity conversation
- **send**: Send a message to Antigravity
- **serve**: Check if Antigravity is being served / accessible
- **status**: Check active CDP connection to Antigravity
- **watch**: Watch for new messages in Antigravity conversation
