工具列表:
- **ask**: Send a prompt and wait for the AI response (macOS ChatGPT desktop app)
- **ax**: Send a prompt and wait for the AI response (ChatGPT ax mode)
- **new**: Start a new ChatGPT conversation
- **read**: Read the current ChatGPT conversation history
- **send**: Send a message to the active ChatGPT conversation
- **status**: Check active CDP connection to ChatGPT Desktop
