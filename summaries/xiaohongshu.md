工具列表:
- creator-note-detail: 小红书单篇笔记详情页数据 (笔记信息 + 核心/互动数据 + 观看来源 + 观众画像 + 趋势数据)
- creator-notes-summary: 小红书最近笔记批量摘要 (列表 + 单篇关键数据汇总)
- creator-notes: 小红书创作者笔记列表 + 每篇数据 (标题/日期/观看/点赞/收藏/评论)
- creator-profile: 小红书创作者账号信息 (粉丝/关注/获赞/成长等级)
- creator-stats: 小红书创作者数据总览 (观看/点赞/收藏/评论/分享/涨粉，含每日趋势)
- download: 下载小红书笔记中的图片和视频
- feed: 小红书首页推荐 Feed (via Pinia Store Action)
- notifications: 小红书通知 (mentions/likes/connections)
- publish: 小红书发布图文笔记 (creator center UI automation)
- search: 搜索小红书笔记
- user: Get public notes from a Xiaohongshu user profile
