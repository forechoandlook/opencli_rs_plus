name: doubao-app
description: 豆包桌面应用对话，支持消息读写、截图、DOM dump
version: 1.0.0
---
工具列表:
- ask: Send a message to Doubao desktop app and wait for the AI response
- dump: Dump Doubao desktop app DOM and snapshot to /tmp for debugging
- new: Start a new chat in Doubao desktop app
- read: Read chat history from Doubao desktop app
- screenshot: Capture a screenshot of the Doubao desktop app window
- send: Send a message to Doubao desktop app
- status: Check CDP connection to Doubao desktop app
