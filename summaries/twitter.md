工具列表:
- accept: Auto-accept DM requests containing specific keywords
- article: Fetch a Twitter Article (long-form content) and export as Markdown
- block: Block a Twitter user
- bookmark: Bookmark a tweet
- bookmarks: Fetch Twitter/X bookmarks
- delete: Delete a specific tweet by URL
- download: 下载 Twitter/X 媒体（图片和视频）
- follow: Follow a Twitter user
- followers: Get accounts following a Twitter/X user
- following: Get accounts a Twitter/X user is following
- hide-reply: Hide a reply on your tweet (useful for hiding bot/spam replies)
- like: Like a specific tweet
- notifications: Get Twitter/X notifications
- post: Post a new tweet/thread
- profile: Fetch a Twitter user profile (bio, stats, etc.)
- reply-dm: Send a message to recent DM conversations
- reply: Reply to a specific tweet
- search: Search Twitter/X for tweets
- thread: Get a tweet thread (original + all replies)
- timeline: Fetch Twitter timeline (for-you or following)
- trending: Twitter/X trending topics
- unblock: Unblock a Twitter user
- unbookmark: Remove a tweet from bookmarks
- unfollow: Unfollow a Twitter user
